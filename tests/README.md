# Тестирование

## Инструкция по запуску

Можно использовать тот же venv, что и для запуска сервиса. Также необходимо установить дополнительные библиотеки:

```
python -m pip install requirements.txt
```

В `pyproject.toml` настроен запуск `pytest`, для тестирования с указанием покрытия достаточно команды:

```
python -m pytest
```

Для нагрузочного тестирования написан `locustfile.py`

```
locust -f tests/locustfile.py
```

## Результаты тестирования

Результат выполнения `python3 -m pytest --cov=app --cov-report=html` можно увидеть в `index.html`, я решила попробовать добиться 100% покрытия

Нагрузочное тестирование при деплое с помощью бесплатных ресурсов render.com показало, что сервис способен выдерживать RPS ~3  при 10 одновременно активных юзерах.
